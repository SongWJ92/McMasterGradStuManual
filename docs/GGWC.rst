社团的活动
----------------------------------------------
GGWC的活动主要有两种形式：Lecture和Technical Session。

- Lecture涵盖的主题包括：

  - 某具体技术岗位的面试技巧和经验，或者工作体会
  - 某项具体技术的知识讲座
  - 创业故事以及经验总结

Lecture主要是邀请具有某项技术特长的在读学生或者已经工作的业界技术大牛来给大家开讲。

- Technical Session

  - 主题主要是业界目前流行的具体技术以及它的具体应用（如机器学习、计算机视觉、数据挖掘、机器人控制等）。
  - 具体运作是由某项技术特长的技术牛带领2-5人做具体的技术训练或者项目。
